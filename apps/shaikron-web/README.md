# Shaikron Web (Evolia)

> ⚠️ **AVISO DE REBRANDING (Evolia by Shaikron)**
> O produto final para o cliente foi oficialmente renomeado para **Evolia**. 
> Toda a identidade visual (textos, títulos, mensagens) no frontend exibe o nome "Evolia". 
> No entanto, por decisão arquitetural de segurança e para não quebrar integrações (Stripe, Appmax, Webhooks, Docker, etc), a infraestrutura raiz e o nome das pastas continuam como **Shaikron** (a Holding/Tecnologia por trás do app).

## Estrutura do App
Este é o frontend principal da plataforma, construído em React/Vite. Ele consome a API do Shaikron (`apps/api`) e interage com o Supabase para Autenticação.

## Tecnologias
- React + Vite
- TailwindCSS
- Supabase (Auth)
- Lucide Icons
